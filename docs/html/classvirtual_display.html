Detailed Description

This is singleton class based on the memory model.

Defines a two-dimensional array virtual map using the fields alias type.

Since this class is Singleton and based on concept lazy initialization, the instance is initialized only upon the first usage of 'getInstance'.

User methods:

  • 'getInstance' - Create and retrieves the Singleton instance.
  • 'getMap' - Provides access to the map pointer array for display purposes.
  • 'put' - Sets data on the virtual map.
